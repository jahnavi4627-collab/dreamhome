# **App Name**: Dream Home Depot

## Core Features:

- User Authentication: Secure user and supplier login using Firebase Authentication.
- Material Listing: Browse and search construction materials from various verified suppliers.
- Material Comparison: Compare different construction materials based on price, specifications, and supplier.
- Shopping Cart and Order Placement: Add materials to a cart and place orders with selected suppliers. User information, materials ordered, and quantity required should be sent to the supplier in a clearly formatted fashion.
- Order Tracking: Track the status and delivery of placed orders.
- Supplier Dashboard: Suppliers manage orders, update material availability, and provide shipping information.
- Personalized Recommendations: A tool recommends construction materials to builders and contractors based on project requirements.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to evoke a sense of home and stability.
- Background color: Off-white (#FAFAFA), a very lightly desaturated beige, to maintain focus on the content.
- Accent color: Muted blue (#6A89CC) to signal trustworthiness and reliability.
- Body font: 'PT Sans', a humanist sans-serif, for readability and a modern touch.
- Headline font: 'Playfair', a modern serif, similar to Didot, for elegance, with 'PT Sans' as the body.
- Use simple, recognizable icons to represent material categories and actions.
- Design a clean, grid-based layout for easy navigation and responsiveness.
- Incorporate subtle transitions to enhance user experience during loading and interactions.
